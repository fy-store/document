## 语义化结构元素

这些语义化结构元素由html5提出

为了保证向后兼容需将该些元素的 display 设置为 block

以下元素默认为块盒



### header

通常用于表示页头，也可以用于表示文章的头部

```html
<header></header>
```



### footer

通常用于表示页脚，也可以用于表示文章的尾部

```html
<footer></footer>
```



### article

通常用于表示整篇文章

```html
<article></article>
```



### section

通常用于表示文章的章节

```html
<section></section>
```



### aside

通常用于表示侧边栏

```html
<aside></aside>
```

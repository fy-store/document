## 关于路径

> 绝对路径
>
> 当跳转目标和当前页面的协议相同时，可以省略协议
>
> 协议名://主机名:端口号/路径

- schema://host:port/path
- 协议名：http https file
- 主机名：域名 IP
- 端口号：如果协议是http，默认端口号80；如果协议是https协议，默认端口号为443

> 相对路径
>
> 以 ./ 开头，./ 可以省略

- ./ 表示当前的资源所在的目录为当前目录
- ../ 表示返回上一级目录
- ../../ 上上一级，之后往上同理叠加
- / 下一级，例如：./a/b/index.html 当前路径下的 a文件夹里面的b文件夹里的 imdex.html 文件

## @ 规则

> @ 规则/指令

<a href="https://developer.mozilla.org/zh-CN/docs/Web/CSS/At-rule" target="_blank">MDN文档</a> 

在css文件中使用：

at-rule称呼: @规则、@语句、css语句、css指令

- import

  @inport "路径";   导入另一个css文件

- charset

  @charset "utf-8";

​		告诉浏览器该css文件，使用的字符编码集是utf-8，必须放在css文件中的第一行

- @media

  媒体查询

- @font-face

  自定义字体

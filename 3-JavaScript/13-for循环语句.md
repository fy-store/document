## for循环语句

第一步声明变量 , 第二步判断表达式是否成立, 如果为 `true` 则执行循环体, 为 `false` 则终止循环



**示例：**

```
var num = 0;
for(var i = 0; i < 10; i++) {
    // code...
    num++
}
```



*解析*

1. 第一步：声明变量 i 赋值为 0；
2. 第二步：判断 i 是否小于 10；
3. 第三步：执行代码体；
4. 第四步：i ++；
5. 第五步：判断 i 是否小于 10；
6. 第六步：i ++；



*变形（类似while循环）：*

- 变量声明放在语句外，变化值放在执行体

```
// 示例：
var i = 0;
for(; i < 10;) {
    // code...
    i++;
}
```



*无限循环*

变量声明、条件表达式和循环后表达式都不是必需的

```js
for(;;) {
    // code...
}
```

此时便成了死循环, 可以在循环体内通过判断使用 break 来进行跳出循环
